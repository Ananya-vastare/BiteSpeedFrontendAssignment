# BiteSpeed Frontend Assignment

A React-based flow builder application built using React Flow (@xyflow/react).  
This project allows users to create, connect, and edit message nodes visually.
Hosted URL : https://bitespeedassingment.netlify.app/

##  Features

- Drag and drop message nodes
- Connect nodes with edges
- Restrict multiple outgoing connections from a single source
- Edit node message dynamically
- Validation before saving flow
- Clean UI with background grid and controls

---

##  Tech Stack

- React
- Vite
- React Flow (@xyflow/react)
- CSS
